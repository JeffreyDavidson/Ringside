# Documentation Standards

## Required Annotations
- `@param` for method parameters
- `@return` for return types  
- `@throws` for exceptions
- `@var` for complex variables

## Forbidden Annotations
- `@author` - Remove these
- `@since` - Remove these
- `@version` - Remove these

## Optional Annotations
- `@see` for references
- `@example` for code examples
